# 🔍 GitHub Search App

Aplicação em Next.js para busca de usuários do GitHub e visualização de perfil e repositórios.

## 🚀 Stack

- Next.js (App Router)
- React
- TypeScript
- React Bootstrap
- Fetch API (GitHub REST API)

---

## 📦 Instalação

Clone o repositório:

```bash
git clone <repo-url>
cd <nome-do-projeto>

npm install
# ou
yarn
# ou
